Sobre

heyo é um tema responsivo, minimalista e bonito para hugo. Ele foi baseado no Anatole de Alexander Bilz.

Features

  • Imagens de capa
  • Visualizações in the sidebar
  • Índices in the sidebar
  • Callout shortcode
  • Badges shortcode
  • Tema escuro
  • Suporte para CSS e JS customizados
  • Suporte para Multilinguas (inglês e tw-tc, por enquanto)
  • Open Graph support
  • Soon™

Quickstart

Para testar o exampleSite é só:

git clone https://github.com/LucasVadilho/heyo-hugo-theme heyo
cd heyo/exampleSite && hugo server --themesDir ../..

Ao vivo

Dá pra checar uma versão ao vivo em heyo-theme.oioipio.com.
